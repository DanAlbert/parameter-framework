string CSubsystem::getMapping ( std::list< const CConfigurableElement * > &  configurableElementPath) const
virtual

Fetch mapping data of an element.

The mapping is represented as a std::string of all the mapping data (key:value) defined in the context of the element. This method gathers the mapping data found in each Element of the configurableElementPath list to format the resulting std::string.

Parameters
[in]configurableElementPathList of all the ConfigurableElements found that have a mapping. Elements are added at the end of the list, so the root Element will be the last one.
Returns
Formatted std::string of the mapping data
